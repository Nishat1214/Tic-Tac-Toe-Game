# 🎮 Tic Tac Toe Game

A classic two-player Tic Tac Toe game built with **React** and **Vite**. Enjoy the timeless game of X's and O's with a clean, modern interface!

![Tic Tac Toe Game](src/assets/hero.png)

## ✨ Features

- 🎯 **Two-Player Mode** - Play with friends on the same device
- 🏆 **Winner Detection** - Automatically detects winning combinations
- 🤝 **Draw Detection** - Knows when the game ends in a draw
- 🎨 **Modern UI** - Clean, responsive design with smooth animations
- 🔄 **Reset Functionality** - Start a new game anytime
- 📱 **Responsive Design** - Works perfectly on desktop, tablet, and mobile

## 🎮 How to Play

1. Player **O** starts the game
2. Click on any empty box to place your mark (X or O)
3. Take turns with your opponent
4. Get **three in a row** (horizontally, vertically, or diagonally) to win
5. If all 9 boxes are filled with no winner, it's a **draw**
6. Click **"Reset Game"** to play again

## 🛠️ Technologies Used

- **React 18** - UI Library
- **Vite** - Build Tool & Dev Server
- **CSS3** - Styling & Animations
- **JavaScript (ES6+)** - Game Logic
